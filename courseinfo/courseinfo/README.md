1. مشروع React يعمل بدون أخطاء ✅
2. تطبيق معلومات المادة مع المكوّنات: Header, Content, Part, Total ✅
3. المكوّنات تستخدم Props بشكل صحيح ✅
4. تمرين الوظيفة — بطاقة الطالب (اختياري للدرجات الإضافية) ✅

## أسئلة للمناقشة

**1. لماذا نقسم التطبيق إلى مكوّنات صغيرة بدلاً من وضع كل شيء في مكوّن واحد؟**

لأن كل مكوّن يكون مسؤولاً عن جزء واحد فقط، مما يجعل الكود أسهل في القراءة والصيانة وإعادة الاستخدام.

**2. ما الفرق بين `<Hello />` و `<hello />`؟**

`<Hello />` بحرف كبير تعني مكوّن React، أما `<hello />` بحرف صغير فتعني عنصر HTML عادي.

**3. ماذا يحصل إذا نسينا `export default App`؟**

سيظهر خطأ في `main.jsx` لأنه لن يجد المكوّن عند الاستيراد ويتوقف التطبيق.

**4. لماذا نستخدم `{}` أحياناً و `""` أحياناً عند تمرير Props؟**

`""` للقيم النصية الثابتة، و`{}` لأي قيمة JavaScript كالمتغيرات والأرقام والتعبيرات.
